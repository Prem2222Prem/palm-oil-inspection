# palm-oil-inspection
โครงงานต้นแบบเครื่องคัดแยกผลปาล์มด้วยปัญญาประดิษฐ์ โดยใช้สถาปัตยกรรม YOLOv11s เพื่อตรวจจับและจำแนกคุณภาพของผลปาล์ม
ใช้ข้อมูลภาพทั้งหมด 601 รูป ประกอบด้วยภาพผลปาล์ม 454 รูป และภาพฉากหลัง 147 รูป 

แบ่งการตรวจจับเป็น 2 คลาส ได้แก่ Good Palm Fruit (ผลปาล์มดี) และ Spoiled palm fruit (ผลปาล์มเสีย) 

แบ่งสัดส่วนข้อมูลสำหรับการฝึกสอน (Train) 80%, ตรวจสอบ (Validation) 10% และทดสอบ (Test) 10%
การพัฒนาโมเดล (Model Development)

โมเดลถูกฝึกสอนบน Google Colab โดยใช้ทรัพยากรประมวลผล GPU NVIDIA Tesla T4 

ตั้งค่าความละเอียดภาพนำเข้าที่ 640 x 640 พิกเซล, กำหนดจำนวนการฝึกสอน 100 Epochs และขนาด Batch size ที่ 16
การทดสอบและฮาร์ดแวร์ (Testing & Hardware)

รองรับการประมวลผลแบบเวลาจริง (Real-time) ร่วมกับกล้องเว็บแคม 

ระบบประมวลผลอ้างอิงการทำงานบนคอมพิวเตอร์ที่มีหน่วยประมวลผลกราฟิก NVIDIA GeForce RTX 3050 

ผ่านการทดสอบประสิทธิภาพการตรวจจับใน 3 สภาวะแสง ได้แก่ แสงปกติ, แสงสลัว และแสงจ้าหรือแสงสะท้อน
